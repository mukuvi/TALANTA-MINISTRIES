# Talanta Ministries International

The official website of Talanta Ministries International -- a church community committed to the Great Commission: "Go Ye Into All The World."

This website serves as the digital home for our congregation, providing information about our services, ministries, programs, and ways to connect with our church family.

## Pages

- **Home** -- Welcome page with service information, ministry highlights, and an invitation to join us
- **About** -- Our story, mission, vision, core values, and what we believe
- **Ministries** -- Overview of all 14 active ministries serving every age group and need
- **DVBS** -- Daily Vacation Bible Study program details, weekly schedule, and activities
- **FAQs** -- Answers to commonly asked questions about the church
- **Contact** -- Ways to reach out, prayer requests, and a contact form

## Getting Started

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Build for production:

```
npm run build
```

Preview the production build:

```
npm run preview
```

## Deployment

This project includes a `vercel.json` configuration for deployment on Vercel. Simply connect your repository to Vercel and it will handle the rest.
